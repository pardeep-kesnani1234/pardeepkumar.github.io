Gauge Chart Configuration

Create a Worksheet with Measure Values in Columns and Measure Names in Rows and place it on the dashboard. The order of the measure values is important. 1st is your "needle" value, 2nd is your "budget" slice and 3rd is your total.
Once the worksheet has been set up for Category choose Measure Names and Measure Values for the rest of the dropdowns.
Reload the visual after saving it!
